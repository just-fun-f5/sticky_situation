# Mapa de Inputs

### Slime

| Acción                 | Tecla  |
| ---------------------- | ------ |
| saltar                 | W      |
| agacharse              | S      |
| moverse a la derecha   | D      |
| moverse a la izquierda | A      |
| enfocar la pantalla    | Cursor |

| Acción                 | Tecla   |
| ---------------------- | ------- |
| Saltar                 | W       |
| lanzar el gancho       | 1-click |
| posicionar el gancho   | cursor  |
| moverse a la derecha   | D       |
| moverse a la izquierda | A       |
